# CodeClause Data Science Internship

Projects completed during my **Data Science internship at CodeClause**. Each project is self-contained with its own dataset and notebook.

## Projects

### 1. Churn Prediction in Telecom Industry
[`Churn Prediction in Telecom Industry/`](./Churn%20Prediction%20in%20Telecom%20Industry/)

A **Logistic Regression** model that predicts customer churn for a telecom provider — identifying the factors driving churn and the most at-risk customer segments. Handles **class imbalance** via upsampling and uses feature engineering on categorical variables.
- **Stack:** pandas, NumPy, matplotlib, seaborn, scikit-learn
- **Dataset:** [Telco Customer Churn (Kaggle)](https://www.kaggle.com/datasets/blastchar/telco-customer-churn)

### 2. Market Basket Analysis
[`Market Basket Analysis/`](./Market%20Basket%20Analysis/)

Uses the **Apriori algorithm** to mine association rules from retail transaction data — surfacing frequently co-purchased items to inform cross-selling and store layout.
- **Stack:** pandas, mlxtend (Apriori), matplotlib
- **Dataset:** [Online Retail (UCI)](https://archive.ics.uci.edu/ml/machine-learning-databases/00352/)

## Skills demonstrated

- Supervised classification with imbalanced data
- Unsupervised pattern mining (association rules)
- End-to-end workflow: preprocessing → feature engineering → modelling → interpretation

Each subfolder contains a detailed README and a runnable Jupyter notebook.
